# Microsoft.UI.Windowing.AppWindowTitleBar.IsVisible

<!--
public bool IsVisible { get; }
-->

## -description

Gets a value that specifies whether this title bar is visible.

## -property-value

`true` if this title bar is visible; otherwise, `false`.
