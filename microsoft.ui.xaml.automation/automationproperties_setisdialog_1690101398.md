# Microsoft.UI.Xaml.Automation.AutomationProperties.SetIsDialog

## -description

Sets a Boolean value that indicates whether the specified element should be identified as a dialog window.

## -parameters

### -param element

The object to identify as a dialog window.

### -param value

**true** if the element should be identified as a dialog window; otherwise, **false**.

## -remarks

[Flyout](..\windows.ui.xaml.controls\flyout.md) and [ContentDialog](..\windows.ui.xaml.controls\contentdialog.md) elements default to **true** for [AutomationProperties.IsDialog](/windows/winui/api/microsoft.ui.xaml.automation.automationproperties#xaml-attached-properties).
