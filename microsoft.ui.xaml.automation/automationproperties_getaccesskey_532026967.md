# Microsoft.UI.Xaml.Automation.AutomationProperties.GetAccessKey

## -description
Gets the value of the [AutomationProperties.AccessKey](/windows/winui/api/microsoft.ui.xaml.automation.automationproperties#xaml-attached-properties) attached property for the specified [DependencyObject](../microsoft.ui.xaml/dependencyobject.md).

## -parameters
### -param element
The [DependencyObject](../microsoft.ui.xaml/dependencyobject.md) to check.

## -returns
The access key, as a string.
