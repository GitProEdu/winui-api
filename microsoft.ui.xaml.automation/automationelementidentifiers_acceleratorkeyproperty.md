# Microsoft.UI.Xaml.Automation.AutomationElementIdentifiers.AcceleratorKeyProperty

## -description
Identifies the accelerator key automation property. The accelerator key property value is returned by the [GetAcceleratorKey](../microsoft.ui.xaml.automation.peers/automationpeer_getacceleratorkey_970307487.md) method.

## -property-value
