# Microsoft.UI.Xaml.Automation.AutomationElementIdentifiers.FullDescriptionProperty

## -description
Identifies the full description automation property.

## -property-value
The automation property identifier.

## -remarks
A localized string that describes the visual appearance or contents of something such as an image or image control is returned by the [GetFullDescription](automationproperties_getfulldescription_1923509144.md) method.
