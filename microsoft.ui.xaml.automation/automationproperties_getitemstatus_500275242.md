# Microsoft.UI.Xaml.Automation.AutomationProperties.GetItemStatus

## -description
Gets the value of the [AutomationProperties.ItemStatus](/uwp/api/microsoft.ui.xaml.automation.automationproperties#xaml-attached-properties) attached property for the specified [DependencyObject](../microsoft.ui.xaml/dependencyobject.md).

## -parameters
### -param element
The [DependencyObject](../microsoft.ui.xaml/dependencyobject.md) to check.

## -returns
The item status of the element.
