# Microsoft.UI.Xaml.RectHelper.GetIsEmpty

## -description

 Returns whether a specified [Rect](/uwp/api/windows.foundation.rect) is equivalent to an [Empty](recthelper_empty.md) [Rect](/uwp/api/windows.foundation.rect). C# and Microsoft Visual Basic code should use [Rect.IsEmpty](/dotnet/api/windows.foundation.rect.isempty) instead.

## -parameters

### -param target

The [Rect](/uwp/api/windows.foundation.rect) to evaluate.

## -returns

 **true** if the specified [Rect](/uwp/api/windows.foundation.rect) is equivalent to an [Empty](recthelper_empty.md) [Rect](/uwp/api/windows.foundation.rect); otherwise, **false**.
