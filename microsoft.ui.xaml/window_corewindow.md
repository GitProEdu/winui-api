# Microsoft.UI.Xaml.Window.CoreWindow

## -description
Gets the [CoreWindow](https://docs.microsoft.com/uwp/api/Windows.UI.Core.CoreWindow) associated 
with this Window. In a Desktop app this is always null.

## -property-value
A **CoreWindow** object.
