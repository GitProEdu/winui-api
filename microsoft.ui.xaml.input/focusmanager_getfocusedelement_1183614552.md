# Microsoft.UI.Xaml.Input.FocusManager.GetFocusedElement

## -description

Retrieves the element in the UI that has focus.

## -returns

The object that has focus. Typically, this is a [Control](../microsoft.ui.xaml.controls/control.md) class.
