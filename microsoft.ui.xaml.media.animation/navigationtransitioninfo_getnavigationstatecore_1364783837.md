# Microsoft.UI.Xaml.Media.Animation.NavigationTransitionInfo.GetNavigationStateCore

## -description
When implemented in a derived class, gets the navigation state string that is reported for navigation actions through [Frame.Navigate](../microsoft.ui.xaml.controls/frame_navigate_442648331.md) and similar API.

## -returns
The string to use for navigation state info.
