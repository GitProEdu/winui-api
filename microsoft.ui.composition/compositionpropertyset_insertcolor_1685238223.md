# Microsoft.UI.Composition.CompositionPropertySet.InsertColor

## -description
Inserts a [Color](/uwp/api/windows.ui.color) key-value pair.

## -parameters
### -param propertyName
The key associated with the value. This key can be used to retrieve the value.

### -param value
The value to insert.
