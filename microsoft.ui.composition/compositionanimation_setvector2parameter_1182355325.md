# Microsoft.UI.Composition.CompositionAnimation.SetVector2Parameter

## -description
Sets a [Vector2](/uwp/api/windows.foundation.numerics.vector2) value parameter for use with an [ExpressionAnimation](expressionanimation.md) or an [expression keyframe](keyframeanimation_insertexpressionkeyframe_876845367.md).

## -parameters
### -param key
The name of the parameter to set.

### -param value
The [Vector2](/uwp/api/windows.foundation.numerics.vector2) value.
