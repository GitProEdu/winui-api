# Microsoft.UI.Composition.CompositionPropertySet.TryGetVector2

## -description
Retrieves the specified [Vector2](/uwp/api/windows.foundation.numerics.vector2) property value.

## -parameters
### -param propertyName
The name of the property to retrieve.

### -param value
If the operation is successful, this parameter will contain the specified [Vector2](/uwp/api/windows.foundation.numerics.vector2) property value when the method returns.

## -returns
Returns a [CompositionGetValueStatus](compositiongetvaluestatus.md) value indicating the outcome of retrieving the key-value pair.
