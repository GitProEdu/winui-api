# Microsoft.UI.Composition.CompositionAnimation.SetColorParameter

## -description
Sets a [Color](/uwp/api/windows.ui.color) value parameter for use with an [ExpressionAnimation](expressionanimation.md) or an [expression keyframe](keyframeanimation_insertexpressionkeyframe_876845367.md).

## -parameters
### -param key
The name of the parameter to set.

### -param value
The [Color](/uwp/api/windows.ui.color) value.
