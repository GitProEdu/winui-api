# Microsoft.UI.Input.PointerDeviceType

<!--
public enum PointerDeviceType
-->

## -description

Specifies the pointer-based input device types recognized by Windows.

## -enum-fields

### -field Touch: 0

Touch pointer.

### -field Pen: 1

Pen pointer.

### -field Mouse: 2

Mouse pointer.

### -field Touchpad: 3

Touchpad pointer.
