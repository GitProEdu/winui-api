# Microsoft.UI.Input.ManipulationCompletedEventArgs.Velocities

<!--

## -description

Gets values that indicate the velocities of the transformation deltas (translation, rotation, scale) for a manipulation at the [ManipulationCompleted](gesturerecognizer_manipulationcompleted.md) event.

## -property-value

The velocities of the accumulated transformations since a [ManipulationStarted](gesturerecognizer_manipulationstarted.md) event.
