# Microsoft.UI.Xaml.Controls.Primitives.AppBarButtonTemplateSettings

## -description

Provides calculated values that can be referenced as **TemplatedParent** sources when defining templates for an [AppBarButton](../microsoft.ui.xaml.controls/appbarbutton.md) control. Not intended for general use.
