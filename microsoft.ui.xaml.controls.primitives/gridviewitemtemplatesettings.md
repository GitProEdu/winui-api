# Microsoft.UI.Xaml.Controls.Primitives.GridViewItemTemplateSettings

## -description
Provides calculated values that can be referenced as **TemplatedParent** sources when defining templates for a [GridView](../microsoft.ui.xaml.controls/gridview.md) control. Not intended for general use.
