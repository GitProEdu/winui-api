# Microsoft.UI.Xaml.Controls.Primitives.AppBarToggleButtonTemplateSettings

## -description

Provides calculated values that can be referenced as **TemplatedParent** sources when defining templates for an [AppBarToggleButton](../microsoft.ui.xaml.controls/appbartogglebutton.md) control. Not intended for general use.
