# Microsoft.UI.Text.RichEditTextDocument.SetText

## -description

Sets the text of the document.

## -parameters

### -param options

Options that control how the document text is set.

### -param value

The new text.

## -remarks

If the string is **NULL**, the text in the document is deleted.
