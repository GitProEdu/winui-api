# Microsoft.UI.Xaml.Data.ICustomPropertyProvider.Type

## -description
Gets the underlying type of the custom property.

## -property-value
The underlying type, with relevant information as the values of the [TypeName](/uwp/api/windows.ui.xaml.interop.typename) structure. [TypeName](/uwp/api/windows.ui.xaml.interop.typename) provides the infrastructure such that property backing does not have to take a dependency on the common language runtime (CLR) and **System.Type**.
